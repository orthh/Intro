# PersonalRecord

개인기록 서비스

URL : http://3.39.239.240/

CI/CD 구현완료

구현중 : redis외 구현완료

백엔드 : SpringBoot + MyBatis + AWS RDS(mariaDB) + redis + Swagger + SpringSecurity
<br/>
서버(CI/CD) : AWS EC2 + jenkins + Docker + nginx + Github Webhook
<br/>
프론트 엔드 : Reactjs + typescript + tailwincss
